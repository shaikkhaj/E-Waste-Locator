# E-Waste-Locator

## Project Overview
E-Waste-Locator is a simple web application designed to help users:
- Locate nearby e-waste recycling facilities
- Learn about the dangers and harmful elements found in electronic waste
- Understand the importance of proper e-waste disposal
- Access resources and information on e-waste management

### Why Use This Project?
Improper disposal of electronic waste can release hazardous substances such as lead, mercury, cadmium, and brominated flame retardants into the environment. These elements are harmful to human health and ecosystems. This project aims to:
- Educate users about the risks associated with e-waste
- Provide guidance on safe disposal and recycling
- Raise awareness about sustainable practices

By using E-Waste-Locator, you can make informed decisions about handling your old electronics and contribute to a cleaner, safer environment.

## How to Run Locally
1. Make sure you have PHP installed.
2. Open a terminal in the project directory.
3. Start the PHP development server:
	```bash
	php -S localhost:8000
	```
4. Open your browser and go to [http://localhost:8000](http://localhost:8000)
